1) Difference Between null and undefined in JavaScript?
Ans: undefined means a variable has been declared but has not been assigned a value yet.

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Ans: map() loops through an array and returns a new modified array.
forEach() loops through an array but does not return anything (it returns undefined).

3) What is the difference between == and ===?
Ans : == compares only values and does type conversion if needed.
=== compares both value and type without type conversion

4) What is the significance of async/await in fetching API data?
Ans : async/await makes asynchronous API calls look like normal synchronous code, which makes the code cleaner and easier to read.

5)  Explain the concept of Scope in JavaScript (Global, Function, Block).
Ans : global scope ----> A variable declared outside any function or block can be accessed anywhere.
Function ---> A variable declared inside a function can only be accessed inside that function.
Block scope ----> Variables declared with let or const inside {} are only accessible inside that block.
